# Running a Dev Server

1. Install Cargo
    - Use [Rustup](https://rustup.rs/)
    - Build from [source](https://github.com/rust-lang/cargo?tab=readme-ov-file#compiling)
2. Install [NPM](https://nodejs.org/en/download)

**1st Terminal**
> ```bash
> cd chatly
> cargo r
> ```

**2nd  Terminal**
> ```bash
> cd chatly
> cd chatly-web
> npm i
> npm run dev
> ```
